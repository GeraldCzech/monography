# /theory-map

## Purpose

Generate or update visual theory maps from BEBA knowledge objects.

## Usage

`/theory-map <concept-or-directory>`

## Instructions

1. Read `02_Theory/Knowledge/Relationships/`.
2. Read relevant concept files.
3. Use the controlled relationship vocabulary.
4. Generate Mermaid diagrams only; do not replace prose theory.

## Output

Create figures in:

`08_Figures/theory_maps/`

## Suggested Outputs

- Mermaid causal chain
- BEBA layer map
- competing mechanisms map
- model comparison map

## Rule

Figures are views on the knowledge base. They should not introduce new theory without accompanying claims.
