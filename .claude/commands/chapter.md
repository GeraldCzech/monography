# /chapter

## Purpose

Develop or revise a dissertation chapter using the BEBA knowledge base.

## Usage

`/chapter 05_Chapters/<chapter>.md`

## Instructions

1. Read `CLAUDE.md`.
2. Read `.claude/agents/writing.md` if present, otherwise use Scientific Editor logic.
3. Read the relevant Research Compass file.
4. Read relevant concept and claim files.
5. Improve the chapter as an argument, not as a literature list.

## Required Chapter Logic

1. Scientific question
2. Problem
3. Theoretical landscape
4. Mechanism
5. Competing explanations
6. Boundary conditions
7. Implications for BEBA
8. Transition to next chapter

## Rule

Do not add unverifiable citations. Do not invent empirical numbers.
