# BEBA Knowledge Base

This directory adapts the monograph repository to the BEBA information model.

## Purpose

The knowledge base stores reusable scientific knowledge independently of chapters and papers.

## Structure

```text
Concepts/          concept-level knowledge objects
Claims/            atomic scientific claims
Evidence/          evidence maps and source-linked observations
Theory_Integration/ reflections on how concepts change BEBA
Ontology_Candidates/ proposed ontology entries awaiting review
```

## Rule

Concepts and claims are the knowledge base. Chapters and papers are outputs.

## Current Status

Initial migration from the previous Markdown ontology has begun with core BEBA concepts:

- Epistemic Uncertainty
- Brand Equity
- Brand Trust
- Brand Commitment

Further concepts should be added through the literature extraction pipeline.
