# Evidence Objects

Evidence objects document empirical or theoretical observations that support, contradict, qualify or refine claims.

## Structure

Evidence is grouped by concept code or source review.

```text
Evidence/
  WG/        Warm Glow evidence
  TR/        Trust evidence
  BE/        Brand Equity evidence
```

Each evidence object should describe:

- source
- finding
- method
- concept(s)
- claim(s) supported or contradicted
- evidence strength
- limitations

## Rule

Evidence is not interpretation. Interpretation belongs in Theory Integration.
