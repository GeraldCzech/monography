## Statement

The sequential, hierarchical structure of the brand equity pyramid (Identity → Meaning → Responses → Resonance) is a valid model for nonprofit, health-promotion organizations.

## Implications for BEBA

This validates using sequential brand-building models as a core component of the brand-enabled behaviour architecture in a nonprofit context.

## Notes

The study tested this on a single, well-established national nonprofit (ParticipACTION) in Canada using cross-sectional survey data.
