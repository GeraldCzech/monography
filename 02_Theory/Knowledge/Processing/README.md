# Processing

Processing files track the movement of source reviews through the BEBA knowledge pipeline.

## Pipeline Status

```text
raw -> markdown -> synthesized -> evidence -> claims -> theory_integration -> ontology_candidate -> reviewed -> integrated
```

## Review Metadata

Each review may receive a metadata file containing:

- review id
- source file
- current status
- concepts detected
- generated artefacts
- warnings
- next action

## Rule

Processing metadata is operational. It does not define theory.
