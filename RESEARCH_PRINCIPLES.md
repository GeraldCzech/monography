# RESEARCH_PRINCIPLES

1. Theory before models.
2. Behaviour before brands.
3. Explanation before prediction.
4. Competing explanations over single models.
5. Evidence before preference.
6. Replication before novelty.
7. Cumulative science.
8. Transparency over convenience.
9. Git is the source of truth.
10. One canonical definition per construct.
11. Literature is evidence, not decoration.
12. AI supports science but does not replace scientific responsibility.
13. Reproducibility by default.
14. Simplicity over complexity.
15. BEBA may evolve, but changes must be documented.
